## ios-dialer

### a cordova plugin, to call the phone dialer
### warning: version 0.5.7 is not tested

* to make phone call
** PhoneDialer.call(phone_number, onSuccess, onError);

* to open the dial
** PhoneDialer.dial(phone_number, onSuccess, onError);
